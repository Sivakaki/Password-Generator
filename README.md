# Password Generator

A simple and interactive Password Generator built using **HTML, CSS, and JavaScript**. This application allows users to generate different types of passwords with a single click.

## Features

* Generate **Weak Passwords**
* Generate **Strong Passwords**
* Generate **Very Strong Passwords**
* Generate **Funny Passwords**
* Clean and responsive user interface
* Instant password generation

## Password Types

### Weak Password

* Consists of lowercase letters only
* Easy to remember
* Low security

**Example:**

```
abcxyz
```

### Strong Password

* Includes uppercase letters
* Includes lowercase letters
* Includes numbers
* Includes special characters
* Characters are shuffled for better randomness

**Example:**

```
A@k8pqwz#
```

### Very Strong Password

* Uses all available character sets
* Long password length
* Highly random
* Suitable for important accounts

**Example:**

```
Q@7mLp#2zX!4kR9$wT
```

### Funny Password

* Uses funny random words
* Includes numbers and symbols
* Easy to remember

**Example:**

```
Monkey42@
Potato17#
Ninja88$
```
